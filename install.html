What They See

Privacy & Security Scanner — Installation Guide

Chrome Extension • Manifest V3
1

Download the Extension

Download or clone this repository to your computer. You can download it as a ZIP file from GitHub and extract it, or clone it using Git.

git clone https://github.com/YOUR_USERNAME/what-they-see.git click to copy
Tip: Make sure you remember where you saved the folder — you'll need to browse to it in Step 3.
2

Open Chrome Extensions Page

Open Google Chrome and navigate to the Extensions management page. You can do this by typing the following URL in your address bar:

chrome://extensions/ click to copy
🔧
Chrome Extensions Page
3

Enable Developer Mode

In the top-right corner of the Extensions page, you'll see a toggle labeled "Developer mode". Turn it ON.

Important: Developer mode must be enabled to install unpacked extensions. This is completely safe and is required for any extension not from the Chrome Web Store.
⚙️
Toggle: Developer mode → ON
4

Load the Extension

After enabling Developer mode, three new buttons will appear at the top. Click "Load unpacked" and a file picker will open.

Navigate to the folder where you saved this extension (the folder containing manifest.json, popup.html, etc.), select the folder, and click "Select".

What to look for: The folder you select should contain these files:
manifest.json  |  popup.html  |  popup.js  |  background.js  |  content.js  |  icon16.png
5

Pin the Extension

The extension is now installed! To make it easily accessible, click the puzzle piece icon 🧩 in Chrome's toolbar, find "What They See", and click the pin icon 📌 to pin it to your toolbar.

📌
Click puzzle icon → Pin the extension
6

Start Scanning

Navigate to any website, click the "What They See" icon in your toolbar, and press "Start Security Scan". The extension will analyze the page and show you a detailed privacy and security report.

That's it! The extension runs entirely in your browser — no data is sent to external servers. Your browsing stays private.

You're All Set!

The extension is now installed and ready to protect your privacy.
Visit any website and click the extension icon to see what they see about you.
